# ✔ 关于

这是一个图片格式和图片大小转换的Chrome浏览器的拓展工具。

- 操作方法

  ![image](./pic/image.png)

1. 首先只需要选择目标图片。

2. 选择目标的文件格式。

3. 输入目标图片的高度和宽度（高度和宽度是可选项可以不输入）。

4. 点击Convert按钮就可以转换图片的格式和图片大小了。
